# PWA Installation Guide

## Overview
Personal Finance Tracker is now a Progressive Web App (PWA) that can be installed on any device - mobile phones, tablets, or desktop computers. Once installed, it works like a native app with offline capabilities.

## Features
- ✅ **Install on Any Device**: Works on iOS, Android, Windows, Mac, Linux
- ✅ **Offline Access**: Continue using the app without internet connection
- ✅ **App-Like Experience**: Runs in standalone mode without browser UI
- ✅ **Quick Access**: Launch from home screen or desktop
- ✅ **Auto-Updates**: Automatically updates when new versions are available

## Installation Instructions

### On Android (Chrome/Edge)
1. Open the app in Chrome or Edge browser
2. A popup will appear at the bottom: "Install Personal Finance Tracker"
3. Tap **"Install Now"**
4. The app will be added to your home screen
5. Launch it like any other app!

**Alternative Method:**
1. Tap the three-dot menu (⋮) in the browser
2. Select "Add to Home screen" or "Install app"
3. Confirm the installation

### On iOS (Safari)
1. Open the app in Safari browser
2. A card will appear with installation instructions
3. Tap the **Share button** (📤) at the bottom
4. Scroll down and tap **"Add to Home Screen"**
5. Tap **"Add"** to confirm
6. The app icon will appear on your home screen

### On Desktop (Chrome/Edge/Brave)
1. Open the app in your browser
2. Look for the install icon (⊕) in the address bar
3. Click **"Install"**
4. The app will open in its own window
5. Access it from your applications menu or taskbar

**Alternative Method:**
1. Click the three-dot menu in the browser
2. Select "Install Personal Finance Tracker"
3. Confirm the installation

### On Windows
- After installation, the app appears in your Start Menu
- Pin it to taskbar for quick access

### On Mac
- After installation, find it in Applications folder
- Add to Dock for easy access

## Troubleshooting

### Install Prompt Not Showing?
- **Chrome/Edge**: The prompt appears after visiting the site a few times
- **iOS Safari**: Must use the Share button method (no automatic prompt)
- **Already Dismissed**: Clear browser data or use browser menu to install

### Can't Find Installed App?
- **Android**: Check app drawer or home screen
- **iOS**: Check all home screen pages
- **Desktop**: Search in applications/start menu

### App Not Working Offline?
- Make sure you've opened the app at least once while online
- Check if service worker is registered (developer tools)

## Uninstalling

### On Mobile
- Long press the app icon
- Select "Uninstall" or "Remove from Home Screen"

### On Desktop
- **Windows**: Right-click app → Uninstall
- **Mac**: Drag app to Trash
- **Chrome**: Settings → Apps → Remove
